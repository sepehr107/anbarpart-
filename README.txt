راهنمای نصب سامانه به‌صورت PWA (اپلیکیشن نصب‌شدنی روی موبایل)
================================================================

این پوشه شامل نسخه‌ی PWA سامانه مدیریت مرسولات است:

  index.html          فایل اصلی برنامه
  manifest.json        مشخصات اپلیکیشن (نام، آیکون، رنگ)
  service-worker.js    فعال‌سازی حالت آفلاین
  icons/               آیکون‌های برنامه

⚠️ نکته‌ی مهم
برای اینکه گزینه‌ی «نصب / افزودن به صفحه اصلی» در مرورگر ظاهر شود،
مرورگرها الزام دارند این فایل‌ها روی یک آدرس اینترنتی امن (https)
یا سرور محلی (localhost) بارگذاری شوند؛ صرفاً دوبار کلیک‌کردن روی
index.html (باز شدن با file://) برای نصب PWA کافی نیست، هرچند
خود برنامه به همان شکل معمول کاملاً کار می‌کند.

روش‌های ساده برای میزبانی این پوشه:

۱) هاست‌های رایگان استاتیک (پیشنهاد می‌شود):
   - Netlify Drop: به آدرس https://app.netlify.com/drop بروید و
     همین پوشه (یا فایل زیپ) را بکشید و رها کنید؛ در چند ثانیه
     یک آدرس اینترنتی امن (https) دریافت می‌کنید.
   - GitHub Pages: محتوای این پوشه را در یک ریپازیتوری گیت‌هاب
     قرار داده و GitHub Pages را برای آن فعال کنید.
   - Cloudflare Pages / Vercel: مشابه همین روش.

۲) اجرای محلی برای تست (روی همان کامپیوتر):
   با نصب Node.js، در همین پوشه دستور زیر را اجرا کنید:
       npx serve .
   سپس آدرسی مانند http://localhost:3000 باز می‌شود.

بعد از باز کردن آدرس https نهایی در مرورگر گوشی (ترجیحاً Chrome
در اندروید یا Safari در iOS):
   - اندروید/Chrome: از منوی سه‌نقطه گزینه‌ی
     «Add to Home screen» یا «Install app» را بزنید.
   - آیفون/Safari: از منوی Share (اشتراک‌گذاری)
     گزینه‌ی «Add to Home Screen» را بزنید.

پس از نصب، آیکون سامانه روی صفحه اصلی گوشی قرار می‌گیرد و
برنامه به‌صورت تمام‌صفحه (بدون نوار آدرس مرورگر) اجرا می‌شود.

نکته: تمام داده‌های مرسولات، مشتریان و تنظیمات همچنان فقط در
حافظه‌ی مرورگر همان بار اجراست و باید طبق روال قبلی از طریق
بخش «پشتیبان‌گیری» ذخیره و بازیابی شوند؛ نصب PWA فقط ظاهر و
دسترسی برنامه را شبیه یک اپلیکیشن واقعی می‌کند.
